How to Use

Page Structure

The page is divided into 3 parts:

  • The gray question box
  • The blue answer boxes
  • The additional buttons

Using this application

The very first page you will see is the name of the tree in the question box and a button that says start. To begin, press Start.

Then answer each question from the question box by clicking on one of the answers from the answer boxes that most relates to your problem. By the end of the questions, if you problem is not resolved, please submit a Service Request to the .

Buttons

If you need would like to Start the questions over, click Start Over. If you need to go back one question, click Back Last Step. And if you need any help, click on Help.
